## 注意

不支持移动端。

## 功能特点

- 实时文本聊天
- 视频通话
- P2P 连接，无需中央服务器
- 简洁的用户界面

## 在线演示

访问 [在线演示](https://himeditator.github.io/p2p-chatroom/) 体验应用。

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- Vue 3
- TypeScript
- PeerJS
- WebRTC
- Ant Design Vue
- Vite
